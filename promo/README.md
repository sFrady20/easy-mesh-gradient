# Promo Video

A short promo video for [easy-mesh-gradient](../lib/README.md), built with [Remotion](https://www.remotion.dev).

The composition (`Promo`, 1080×1080 @ 30fps) types out a code snippet, reveals a live mesh gradient generated by the library itself, and ends on the install command.

## Commands

```bash
# Open Remotion Studio
bun run dev

# Render the video
bun run render Promo out/promo.mp4

# Bundle
bun run build
```

Run `bun install` from the repo root first — this workspace links `easy-mesh-gradient` from `../lib`, so build the library (`bun run build` at the root) before rendering.
